# python-all-in-one

Soldesk 교육 과정에서 진행한 Python 실습들을 정리한 저장소입니다.  
기초 문법부터 파일 처리, 예외처리, 함수, 클래스 등 주요 개념을  
폴더별로 단계적으로 연습한 코드들로 구성되어 있습니다.

## 📂 폴더 구성
- `01` : 기본 문법, 변수, 자료형 실습
- `02` : 리스트, 튜플, 딕셔너리, 반복문 등 자료구조 실습
- `04` : 함수, 모듈, 예외 처리 실습
- `06` : 파일 입출력, JSON 처리 등 데이터 관련 실습
- `07` : 클래스, 객체지향 프로그래밍(OOP) 기초
- `08` : 고급 문법 및 추가 예제
- `09` : 프로젝트형 실습 및 종합 예제
- `test` : 테스트 코드 및 임시 파일

## 🎯 목적
- Python 기본 개념을 반복 학습하며 정리
- 실습으로 익힌 코드들을 향후 참고용으로 보관
- 기초부터 중급까지 흐름 있게 정리된 실습 자료 구축
